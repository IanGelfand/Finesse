## Run Locally

Clone the project

```bash
  git clone https://github.com/IanGelfand/Finesse.git
```

Go to the project directory

```bash
  cd Finesse
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
